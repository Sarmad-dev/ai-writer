## Navigation

### Dashboard (`/dashboard`)
Your home base for content management

**What you see**:
- All your content sessions
- Status badges (Pending, Generating, Completed, Failed)
- Preview of each session

**Actions**:
- Click "New Content" to start a new session
- Click any session card to view/edit
- Use navigation to switch to Chat

### Chat (`/chat`)
Simple conversation interface

**What you see**:
- Sidebar with conversation history
- Main chat area
- Input box at bottom

**Actions**:
- Click "New Chat" to start fresh
- Select previous conversations from sidebar
- Type and press Enter to send

### Content Detail (`/content/[id]`)
Full-featured content generation

**What you see**:
- Workflow status (Analyzing, Searching, Generating, etc.)
- Streaming content as it's created
- Approval requests (when needed)

**Actions**:
- Watch the AI work through the workflow
- Approve/reject web search results
- View generated content
- Edit with TipTap editor (coming soon)
